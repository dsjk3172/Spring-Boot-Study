웹 기초지식


서버와 클라이언트의 의미

Server : 서비스를 제공하는 컴퓨터

client : 서비스를 사용하는 컴퓨터


IP와 PORT에 대한 개념

IP : Internet Protocol의 줄임말로, 인터넷상에서 사용하는 주소체계

Port : IP 내에서 애플리케이션 상호 구분(프로세스 구분)을 위해 사용하는 번호


도메인과 DNS의 개념

도메인 : 웹 브라우저를 통해 특정 사이트에 진입을 할 때, IP 주소를 대신하여 사용하는 주소

DNS : Domain Name System의 줄임말로, 호스트의 도메인 네임을 네트워크 주소로 변환하거나, 그 반대의 역할을 수행하는 시스템
